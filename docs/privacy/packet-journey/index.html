パケットの旅 プライバシーポリシー

「パケットの旅」（以下「本アプリ」）は、ネットワークの仕組みをインタラクティブに学べる iOS 教育アプリです。本アプリでは、利用者のプライバシーを尊重し、以下の方針で情報を取り扱います。

1. 本アプリが保存する情報

本アプリは純粋に教育目的のアプリです。利用者が入力する個人情報を収集・保存しません。本アプリが端末内に保存する情報は以下に限られます。

  • オンボーディング画面の表示済みフラグ（アプリの初回起動を記録するための情報）

上記の情報は、利用者の端末内にのみ保存されます。開発者が閲覧・取得することはありません。

2. 外部サーバーへの送信について

本アプリは、ログイン機能・独自サーバー同期・クラウド保存機能を提供していません。本アプリ内のいかなる情報も、開発者のサーバーへ自動送信されることはありません。

3. アクセス解析・広告・トラッキングについて

本アプリは、第三者広告・広告目的のトラッキング・外部解析 SDK を一切使用していません。今後これらの機能を導入する場合は、本プライバシーポリシーを更新し、App Store 上のプライバシー情報も合わせて更新します。

4. 通知について

本アプリは現時点でプッシュ通知・ローカル通知を送信しません。

5. お問い合わせ時に取得する情報

利用者がメール等で問い合わせを行う場合、以下の情報を取得することがあります。

  • メールアドレス
  • 問い合わせ内容
  • 利用端末・iOS バージョン・アプリバージョン
  • 不具合調査に必要な範囲の情報

これらの情報は、問い合わせ対応・不具合調査・機能改善のためにのみ利用します。問い合わせ対応以外の目的で第三者に提供することはありません。

6. 第三者提供について

開発者は、以下の場合を除き、取得した情報を第三者に提供しません。

  • 利用者の同意がある場合
  • 法令に基づき開示が必要な場合
  • 問い合わせ対応や不具合調査のために必要な範囲で、適切な管理のもと外部サービスを利用する場合

7. データの削除について

端末に保存されたデータは、本アプリを端末から削除することで完全に削除されます。

8. プライバシーポリシーの変更

本プライバシーポリシーの内容は、法令の変更・機能追加・運用方針の変更などに応じて改定することがあります。重要な変更がある場合は、アプリ内またはサポートページ上でお知らせします。
