  2. ウィジェット機能との連携はApp Groupsを通じてデバイス内で行われ、外部への送信は行いません。
  3. 本アプリの開発者を含む第三者が、これらのデータにアクセスすることはできません。

  1. 本アプリは、ご利用者のデータを外部サーバーへ送信しません。

本アプリは、以下のiOS機能へのアクセス許可を求める場合があります。それぞれの用途のみに使用し、目的外の利用は行いません。
